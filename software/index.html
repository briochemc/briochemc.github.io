Citable software

  1. GNOM: An optimized steady-state model of the modern global marine neodymium cycle Benoît Pasquier, Sophia K. V. Hines, Hengdi Liang, Yingzhe Wu, Steven L. Goldstein, Seth G. John 2021 doi: 10.5281/zenodo.5651295
  2. F1Method.jl: A julia package to autodifferentiate through a steady-state solver Benoît Pasquier 2019 doi: 10.5281/zenodo.2667835
    This package implements the F-1 algorithm described in Pasquier and Primeau (in preparation). It allows for efficient quasi-auto-differentiation of an objective function defined implicitly by the solution of a steady-state problem.
  3. AIBECS.jl: A tool for exploring global marine biogeochemical cycles Benoît Pasquier, François W. Primeau, Seth G. John 2019 doi: 10.5281/zenodo.2864051
  4. WorldOceanAtlasTools.jl: Download and process World Ocean Data in Julia Benoît Pasquier 2019 doi: 10.5281/zenodo.2677666
  5. HyperDualNumbers.jl Rob J. Goedman, Nicola, Benoît Pasquier, Timothy Holy, Alexander Ronald Altman, Christopher Rackauckas, Tony Kelman, Viral B. Shah, Jishnu Bhattacharya 2018 doi: 10.5281/zenodo.1660762
    Julia implementation of HyperDualNumbers.
  6. HyperDualMatrixTools.jl Benoît Pasquier 2018 doi: 10.5281/zenodo.1670841
    A little Julia module to allow factorization and backslash to work with hyperdual-valued arrays and sparse arrays.
  7. DualMatrixTools.jl Benoît Pasquier 2018 doi: 10.5281/zenodo.1493571
    A little Julia module to allow factorization and backslash to work with dual-valued arrays and sparse arrays.
